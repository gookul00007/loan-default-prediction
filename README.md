# Loan Default Prediction

## Problem
Predict whether a customer will default on a loan.

## Approach
- EDA
- Data cleaning and preprocessing
- Feature engineering
- Random Forest model
- Handling imbalanced data

## Business Impact
This model helps financial institutions identify high-risk customers, enabling better credit decision-making and reducing potential losses.

## Results
- Accuracy: 73%
- Recall (Defaulters): 63%
- F1-score: 0.35

## Tech Stack
Python, Scikit-learn, Pandas, NumPy

## How to Run

1. Clone the repository  
2. Install dependencies:
   pip install -r requirements.txt  
3. Run the notebook in Jupyter/Colab
